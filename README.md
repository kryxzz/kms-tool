# KMS Tool
It's a simple KMS Tool I made. It can deactivate or activate.

# FAQ
Q: **Why does it deactivate?**
A: You might have to run the script everytime it deactivates, which is about 180 days.

Q: **It doesn't work!**
A: Run as Administrator.
